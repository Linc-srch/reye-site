How NVIDIA engineers and researchers build with Codex

How NVIDIA engineers and researchers build with Codex

2026-05-18 · openai_rss

Summary

NVIDIA engineers and researchers use OpenAI's Codex, powered by GPT-5.5, as a default tool for complex engineering and end-to-end machine learning research workflows. Codex runs on NVIDIA GB200/GB300 infrastructure, enabling autonomous long sessions, bug detection, and code translation (e.g., Python to Rust). It has improved research workflow speed by 10x and allowed rapid prototyping, such as building an internal podcast app in hours. Over 40,000 NVIDIA employees have access to Codex, which is used for production system development and automated experimentation.
