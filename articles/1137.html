Databricks brings GPT-5.5 to enterprise agent workflows

Databricks brings GPT-5.5 to enterprise agent workflows

content-management digital-publishing rlhf
Summary

Databricks integrates OpenAI's GPT-5.5 into enterprise agent workflows, achieving state-of-the-art performance on the OfficeQA Pro benchmark. The model reduces errors by 46% compared to GPT-5.4, excelling in parsing scanned PDFs, legacy documents, and long-context tasks. GPT-5.5 demonstrates improved orchestration, avoiding unnecessary search detours and reliably retrieving context. Available through AI Unity Gateway, AgentBricks, and Agent Supervisor API, it enables customers to build custom multi-step agent workflows with enhanced accuracy and efficiency.

摘要

Databricks将OpenAI的GPT-5.5集成到企业agent工作流中，在OfficeQA Pro基准测试上达到最先进水平。相比GPT-5.4，错误率降低46%，在解析扫描PDF、遗留文档和长上下文任务中表现优异。GPT-5.5改进了编排能力，避免不必要的搜索绕路，可靠地检索上下文。通过AI Unity Gateway、AgentBricks和Agent Supervisor API提供，支持客户构建自定义多步agent工作流，提升准确性和效率。
