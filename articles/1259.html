Webworm Deploys EchoCreep and GraphWorm Backdoors Using Discord and MS Graph API

Webworm Deploys EchoCreep and GraphWorm Backdoors Using Discord and MS Graph API

Summary

Cybersecurity researchers have identified new activity from the China-aligned threat actor Webworm in 2025, deploying two custom backdoors: EchoCreep, which uses Discord for command-and-control (C2), and GraphWorm, which leverages Microsoft Graph API. Webworm targets government agencies and enterprises in IT, aerospace, and electric power sectors across Russia, Georgia, Mongolia, and other Asian nations, increasingly focusing on European countries. The group employs stealthy proxy tools and abuses a fake WordPress GitHub repository to host malware. The discovery marks a shift away from traditional RATs toward legitimate services for C2, highlighting the adversary's evolving tactics.

摘要

网络安全研究人员发现，2025年与China关联的威胁组织Webworm部署了两个自定义后门：EchoCreep使用Discord进行C2通信，GraphWorm则利用Microsoft Graph API。Webworm主要针对俄罗斯、格鲁吉亚、蒙古及多个亚洲国家的政府机构和IT、航空航天、电力行业，并逐渐将目标扩展至欧洲。该组织采用更隐蔽的代理工具，并滥用伪造的WordPress GitHub仓库托管恶意软件。这一发现表明其从传统RAT转向利用合法服务进行C2，反映了对手战术的演进。
