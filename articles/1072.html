2) CRGP

C — Context

Expanding LLMs to new languages typically requires costly Continued Pre-Training (CPT) and data-intensive alignment (SFT + RL). Recent data-free merging methods try to bypass alignment by fusing a CPT-enhanced model with its instruct counterpart, but suffer from a fundamental trade-off: preserving original abilities dilutes new language acquisition and vice versa.

R — Related work

  • Data-free merging for language expansion (Yamaguchi et al., 2024; Cao et al., 2025) uses linear averaging or delta merging but suffers from catastrophic forgetting or limited new language gains.
  • Upcycling dense models to MoE (Komatsuzaki et al., 2023) for efficient multi-task or multilingual training.
  • Parameter-efficient fine-tuning and delta tuning (e.g., LoRA, T-Few) for adapting LLMs with minimal data.
  • Mixture-of-Experts architectures in large-scale LLMs (e.g., DeepSeek, Qwen2.5-MoE) for scaling with sparse activation.

G — Gap

Existing data-free merging methods for language expansion face a critical trade-off: linear averaging dilutes CPT updates (limiting new language gains), while delta merging causes drastic weight shifts (harming original language retention). No prior work effectively decouples language-specific knowledge from alignment capabilities without expensive post-training.

P — Proposal

DeltaMoE first upcycles a dense base model into an MoE by freezing the original expert (knowledge anchor) and training new experts and a router on new language data (sparse upcycling). Then, it computes a parameter delta from the original dense instruct model and grafts it onto all MoE experts via MoE delta merging, transferring alignment abilities without instruction data.

4) Experiments

Main results

On expanded language benchmarks, DeltaMoE improves average performance by 1.7 points over delta-based baselines (e.g., Dense-FT-Delta) and preserves original language capabilities by 1.5 points, outperforming methods with similar FLOPs or parameter counts across multiple base models and post-training deltas.

Ablation

Ablation on the number of experts and top-k routing shows that using 4 experts with top-2 routing yields the best trade-off; freezing the original expert is critical for preserving original language performance (e.g., dropping it reduces original language scores by ~2 points).

Limitations

['The method assumes availability of a high-quality CPT corpus for the target languages, which may be scarce for low-resource languages.', 'MoE upcycling increases total parameter count and inference FLOPs (though only top-k experts are activated), potentially limiting deployment on memory-constrained devices.', 'The approach is evaluated on a limited set of base models and post-training deltas; generalization to very large models (e.g., >100B) or non-SwiGLU architectures is not verified.']

5) Why it matters

  1. Provides a data-efficient path to multilingual LLMs by eliminating the need for expensive alignment data (SFT/RL), directly relevant to your work on reducing training costs for low-resource languages.
  2. The MoE delta merging technique is a novel way to transfer alignment capabilities without parameter conflicts, which could inspire similar approaches for other cross-task or cross-domain knowledge transfer.
  3. Demonstrates that sparse upcycling with a frozen expert effectively preserves original knowledge, a key insight for continual learning scenarios where catastrophic forgetting is a concern.
  4. The method's generality across different base models and post-training deltas suggests it could be a plug-and-play module for future multilingual LLM development.
