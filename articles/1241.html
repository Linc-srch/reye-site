LiteSpeed cPanel Plugin CVE-2026-48172 Exploited to Run Scripts as Root

LiteSpeed cPanel Plugin CVE-2026-48172 Exploited to Run Scripts as Root

supply-chain-attack
Summary

A critical privilege escalation vulnerability (CVE-2026-48172, CVSS 10.0) in the LiteSpeed User-End cPanel Plugin is being actively exploited in the wild. The flaw allows any cPanel user, including an attacker or compromised account, to execute arbitrary scripts as root via the lsws.redisAble function. Affecting plugin versions 2.3 to 2.4.4, the issue is patched in version 2.4.5. LiteSpeed has released a bundled update (WHM plugin 5.3.1.0 with cPanel plugin 2.4.7) and provided an indicator of compromise using grep. Users unable to patch immediately are advised to uninstall the plugin. This disclosure follows the active exploitation of another critical cPanel vulnerability (CVE-2026-41940) for deploying Mirai botnet and Sorry ransomware.

摘要

LiteSpeed cPanel插件中存在一个严重权限提升漏洞CVE-2026-48172（CVSS 10.0），正在被积极利用。攻击者可利用lsws.redisAble函数以root权限执行任意脚本。影响版本2.3至2.4.4，已在2.4.5中修复。LiteSpeed发布了包含修复的WHM插件5.3.1.0。建议用户立即升级或卸载插件。此前另一关键cPanel漏洞CVE-2026-41940也被用于部署Mirai僵尸网络和Sorry勒索软件。
