Kunqi Xu, Jitao Li, Jianglong Ye, Tianshu Tang, Isabella Liu, Sifei Liu, Xueyan Zou  ·  2026-05-19  ·  arxiv  ·  Original →

Summary

WorldString proposes a unified neural architecture that learns actionable digital twins of articulated, skinned, and soft objects directly from point clouds or RGB-D video, enabling controllable state-conditioned 3D generation.

摘要

本文提出WorldString，一种统一的神经架构，能够直接从点云或RGB-D视频中学习可操作的数字孪生，涵盖铰接、蒙皮和软体物体。它通过全微分网络实现状态条件化的3D生成，在多个物体类别上取得了最先进的重建精度，为机器人策略学习和物理世界模型提供了基础构建模块。

1) Why-read

WorldString proposes a unified neural architecture that learns actionable digital twins of articulated, skinned, and soft objects directly from point clouds or RGB-D video, enabling controllable state-conditioned 3D generation.

2) CRGP

C — Context

Physical world models aim to provide agents with action-conditioned dynamics for planning and reasoning. Current approaches fall into video generation (high-fidelity but physically inconsistent), neural 3D reconstruction (3D-consistent but struggle with dynamics), and physics simulation (grounded but hard to infer from vision). None explicitly model object state transitions in a unified, differentiable manner.

R — Related work

  • World models as learned latent simulators (Ha & Schmidhuber, 2018) and physical world models as digital twins (Assouel et al., 2025).
  • Dynamic NeRFs (e.g., D-NeRF, Nerfies) and dynamic 3DGS (e.g., 4DGS, Deformable 3DGS) model motion as time-conditioned warps or per-primitive trajectories.
  • Classical object modeling: rigid (meshes, point clouds), articulated (URDF kinematic trees), skinned (LBS), and soft (FEM/continuum mechanics).
  • Video-based digital twins for deformable objects (e.g., PhysGaussian, PAC-NeRF) reconstruct geometry with simulatable parameters.

G — Gap

Existing methods either model dynamics implicitly (generation) or via dense warps/primitive trajectories (reconstruction), lacking a unified, controllable representation that explicitly captures object deformation across articulated, skinned, and soft regimes. No prior work provides a fully differentiable architecture that learns state-conditioned deformations directly from real-world data while enabling integration with policy learning.

P — Proposal

WorldString learns a canonical point cloud and a state-conditioned deformation field via a fully differentiable neural network. It models articulated objects with forward kinematics, skinned objects with linear blend skinning, and soft objects with a learned Jacobian approximation. The architecture is trained end-to-end from point clouds or RGB-D video and outputs a 3D point cloud conditioned on a state vector.

3) Figures

Figure extraction is performed separately and inlined post-generation.

4) Experiments

Main results

WorldString achieves state-of-the-art reconstruction accuracy on articulated objects (Chamfer distance 0.0032 vs. 0.0041 for baseline), skinned objects (CD 0.0045 vs. 0.0053), and soft objects (CD 0.0061 vs. 0.0078). On the Dr.Robot baseline for articulated objects, WorldString reduces CD by 22%. Qualitative results show faithful deformation for unseen joint configurations.

Ablation

Removing the Jacobian-based soft object module increases CD by 34% (from 0.0061 to 0.0082). Using a single shared deformation field instead of category-specific modules increases CD by 18% on articulated objects and 25% on skinned objects.

Limitations

WorldString currently models single objects in isolation, not full scenes with multiple interacting objects. The soft object model relies on a linear approximation, which may fail for large deformations or topological changes. Training requires object-centric video with known state annotations, limiting applicability to unlabeled in-the-wild data.

5) Why it matters

  1. Provides a unified differentiable representation for object dynamics, enabling seamless integration with policy learning and neural dynamics for robotics.
  2. Bridges the gap between 3D reconstruction and physical simulation by learning state-conditioned deformations directly from real data.
  3. Offers a principled building block for physical world models, potentially scaling to scene-level composition.

6) Next steps

  • [ ] Extend WorldString to multi-object scenes with interaction modeling (e.g., contact dynamics).

  • [ ] Integrate with a policy network for closed-loop control tasks (e.g., manipulation).

1) 阅读理由

WorldString提出了一种统一的神经架构，可直接从点云或RGB-D视频中学习铰接、蒙皮和软体物体的可操作数字孪生，实现状态条件化的可控3D生成。

2) CRGP（背景 / 相关工作 / 差距 / 方案）

物理世界模型旨在为智能体提供基于动作的条件动力学以进行规划和推理。现有方法包括视频生成（高保真但物理不一致）、神经3D重建（3D一致但难以处理动力学）和物理模拟（有物理基础但难以从视觉推断），均未以统一、可微分的方式显式建模物体状态转换。相关工作包括作为学习型潜在模拟器的世界模型、动态NeRF和动态3DGS（将运动建模为时间条件扭曲或逐基元轨迹）、经典物体建模（刚体、铰接、蒙皮、软体）以及基于视频的可变形物体数字孪生。现有方法要么隐式建模动力学（生成），要么通过密集扭曲/基元轨迹（重建）建模，缺乏统一、可控的表示来显式捕获铰接、蒙皮和软体物体的变形。尚无工作提供全微分架构，能从真实数据中直接学习状态条件化变形并集成策略学习。WorldString学习一个规范点云和一个通过全微分神经网络实现的状态条件化变形场。它对铰接物体使用正向运动学，对蒙皮物体使用线性混合蒙皮，对软体物体使用学习到的雅可比近似。该架构从点云或RGB-D视频端到端训练，输出条件于状态向量的3D点云。

3) 实验

WorldString在铰接物体上实现了最先进的重建精度（Chamfer距离0.0032，基线为0.0041），在蒙皮物体上（CD 0.0045 vs. 0.0053），在软体物体上（CD 0.0061 vs. 0.0078）。在Dr.Robot铰接物体基线上，WorldString将CD降低了22%。定性结果显示，对于未见过的关节配置，变形结果保真度高。

消融实验

移除基于雅可比矩阵的软体模块后，CD增加了34%（从0.0061升至0.0082）。使用单一共享变形场替代类别特定模块，导致铰接物体CD增加18%，蒙皮物体CD增加25%。

局限性

WorldString当前仅建模孤立单物体，而非包含多个交互物体的完整场景。软体模型依赖线性近似，可能在大变形或拓扑变化时失效。训练需要以物体为中心的视频和已知状态标注，限制了其在无标注野外数据上的适用性。

4) 重要性

  1. 提供了一种统一的、可微分的物体动力学表示，可无缝集成到机器人策略学习和神经动力学中。
  2. 通过直接从真实数据学习状态条件化变形，弥合了3D重建与物理模拟之间的鸿沟。
  3. 为物理世界模型提供了原则性的构建模块，有望扩展到场景级别的组合。

5) 下一步

  • [ ] 结合无监督状态发现技术，从视频中自动提取状态，减少对状态标注的依赖。
