Latent Action Reparameterization for Efficient Agent Inference

用于高效智能体推理的潜在动作重参数化

Wenhao Huang, Qingwen Zeng, Qiyue Chen, Zijie Guo, Yu Sun, Cheng Yang, Siru Ouyang, Jiri Gesi, Fang Wu, Jiayi Zhang, Huaming Chen, Bang Liu, Xiangru Tang, Chenglin Wu

Summary

LAR introduces a new perspective on LLM agent efficiency by reparameterizing the action space into learnable latent actions, reducing decision horizon and inference cost without sacrificing task performance.

摘要

LAR通过从智能体轨迹中学习紧凑的潜在动作空间，将多步语义行为编码为单个潜在动作，从而大幅压缩决策时间步。该方法在LLM智能体基准测试中实现了30-50%的动作令牌减少和20-40%的推理时间缩短，同时保持或提升了任务成功率。这篇论文为提升LLM智能体推理效率提供了全新的视角。

1) Why-read

LAR introduces a new perspective on LLM agent efficiency by reparameterizing the action space into learnable latent actions, reducing decision horizon and inference cost without sacrificing task performance.

2) CRGP

C — Context

LLM agents generate long sequences of low-level textual actions, leading to large effective decision horizons and high inference cost. Prior work focuses on system-level optimizations, prompt engineering, or token-level generation control, but these methods leave the granularity of decision-making unchanged.

R — Related work

  • Prompting and input-level methods (e.g., Chain-of-Thought, concise hint) improve accuracy but increase generation length.
  • Token-level generation control (e.g., TokenSkip) prunes low-utility tokens but does not reduce the number of decision steps.
  • Context and memory optimization (e.g., ACON) compresses interaction histories but does not alter the action representation.
  • Hand-crafted macros and hierarchical controllers exist but are not learned from data and may not generalize.

G — Gap

Existing efficiency methods operate at the token or memory level, leaving the fundamental decision horizon dictated by token-level actions. There is no learned abstraction that collapses multi-step behaviors into a single latent action while preserving executability and expressiveness.

P — Proposal

LAR learns a compact latent action space via reparameterization of agent trajectories. Each latent action encodes a multi-step semantic behavior, and the model is trained to predict and execute these latent actions. Low-entropy structural components (e.g., prompt templates) are abstracted, while high-entropy parameters remain explicit to maintain executability.

4) Experiments

Main results

Across multiple LLM agent benchmarks (including SWE-bench and others referenced in Table 1 & 8), LAR reduces action tokens by 30-50% and wall-clock inference time by 20-40% while maintaining or improving task success rates. The approach transfers to held-out benchmarks without retraining.

Ablation

Ablation on compression ratio reveals a performance collapse threshold at approximately 50% token reduction, beyond which task success rates drop sharply, indicating the boundary between compressible structural redundancy and essential semantic content.

Limitations

['Requires training on agent trajectories, which may not be available for all domains.', 'The learned latent actions may not transfer to environments with very different action structures.', 'The compression threshold is empirical and may vary across tasks and models.']

5) Why it matters

  1. Offers a complementary path to scaling LLM agent inference beyond model architecture and hardware improvements.
  2. Highlights action representation learning as a critical underexplored factor in agent efficiency.
  3. Provides a principled way to reduce decision horizon without ad-hoc macro design.

1) 阅读理由

LAR通过将动作空间重参数化为可学习的潜在动作，在降低决策时间步和推理成本的同时不牺牲任务性能，为LLM智能体效率提供了全新视角。

2) CRGP（背景 / 相关工作 / 差距 / 方案）

背景：LLM智能体生成冗长的低级文本动作序列，导致巨大的有效决策时间步和高推理成本。先前工作集中于系统级优化、提示工程或令牌级生成控制，但这些方法未改变决策的粒度。相关工作：提示和输入级方法（如思维链、简洁提示）提高了准确率但增加了生成长度；令牌级生成控制（如TokenSkip）剪除了低效用令牌但未减少决策步骤数；上下文和记忆优化（如ACON）压缩了交互历史但未改变动作表示；手工设计的宏和层次控制器并非从数据中学习且可能缺乏泛化性。差距：现有效率方法作用于令牌或记忆层面，未能改变由令牌级动作所决定的根本决策时间步。缺乏一种学习到的抽象机制，能将多步行为折叠为单个潜在动作，同时保持可执行性和表达力。方案：LAR通过重参数化智能体轨迹学习一个紧凑的潜在动作空间。每个潜在动作编码一个多步语义行为，模型被训练用于预测并执行这些潜在动作。低熵的结构性成分（如提示模板）被抽象，而高熵的参数保持显式以确保可执行性。

3) 实验

在多个LLM智能体基准测试（包括SWE-bench及表1、表8中引用的其他基准）中，LAR将动作令牌减少30-50%，端到端推理时间降低20-40%，同时任务成功率维持或提升。该方法无需重新训练即可迁移至未见过的基准测试。

消融实验

关于压缩比率的消融实验揭示了一个约50%令牌减少的性能崩溃阈值，超过该阈值后任务成功率急剧下降，表明可压缩的结构性冗余与必需的语义内容之间存在边界。

局限性

['需要在智能体轨迹上进行训练，这可能并非对所有领域都可用。', '学习到的潜在动作可能无法迁移到动作结构差异极大的环境中。', '压缩阈值是经验性的，可能因任务和模型而异。']

4) 重要性

  1. 提供了一条超越模型架构和硬件改进的LLM智能体推理扩展的互补路径。

  3. 提供了一种无需临时宏设计的、有原则的方法来减少决策时间步。
