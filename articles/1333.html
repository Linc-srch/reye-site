Scaling, Benchmarking, and Reasoning of Vision-Language Agents for Mobile GUI Navigation

面向移动GUI导航的视觉语言智能体：规模扩展、基准测试与推理

启示 · Relevance

For AI Security, this work provides a new empirical benchmark and dataset (HyperTrack, GUIEvalKit) to red-team and evaluate the safety of vision-language agents in mobile GUI navigation, directly enabling your research on automated red-teaming and temporal safety evaluation for embodied agents. For Harmony Security and SOC, this paper highlights a critical new attack surface: as on-device GUI agents become more capable via reinforcement-based finetuning, their out-of-domain generalization introduces unpredictable vulnerabilities that must be monitored and benchmarked against existing OS security baselines.

对于AI安全方向，这项工作提供了新的实证基准和数据集（HyperTrack, GUIEvalKit），可用于对移动GUI导航中的视觉语言智能体进行红队测试和安全评估，直接支持你在具身智能体自动化红队和时序安全评估方面的研究。对于Harmony安全与SOC方向，本文揭示了一个关键的新攻击面：随着端侧GUI智能体通过强化学习微调变得更强大，其跨域泛化能力会引入不可预测的漏洞，必须被持续监控并与现有操作系统安全基线进行对标。

Summary

This paper systematically studies data scaling, benchmarking, and reasoning for VLM-based mobile GUI agents, introducing a large-scale dataset (HyperTrack) and a unified evaluation toolkit (GUIEvalKit) to show that reinforcement-based finetuning outperforms supervised finetuning, especially in out-of-domain settings.

摘要

本文系统研究了基于视觉语言模型（VLM）的移动GUI智能体在数据规模扩展、基准测试和推理方面的关键问题。作者构建了包含超过16,000个真实世界任务、覆盖674个中文移动应用的大规模数据集HyperTrack，并推出了统一离线评测工具包GUIEvalKit。实验表明，基于强化学习的微调（DAPO风格RL）在域内和域外任务上均显著优于监督微调，且性能随训练数据量的对数近似线性增长。

1) Why-read

This paper introduces HyperTrack and GUIEvalKit to systematically study data scaling, benchmarking, and reasoning for VLM-based mobile GUI navigation, showing that reinforcement-based finetuning outperforms supervised finetuning especially with larger data.

2) CRGP

C — Context

Vision-Language Models (VLMs) have advanced rapidly in mobile GUI navigation, but existing benchmarks are limited in scale and diversity, and the interplay between data scaling and training methods (supervised vs. reinforcement learning) remains underexplored. Prior work lacks standardized toolkits for offline evaluation across diverse apps.

R — Related work

  • Existing mobile GUI navigation datasets (e.g., AndroidEnv, AppBuddy) are smaller and often limited to English apps.
  • Prior work on VLM agents for GUI tasks (e.g., ScreenAgent, CogAgent) focuses on model architecture rather than systematic scaling and benchmarking.
  • Reinforcement learning for GUI agents has been explored but not systematically compared with supervised finetuning under varying data scales.

G — Gap

Current mobile GUI navigation research lacks a large-scale, diverse dataset covering real-world Chinese apps and a unified benchmarking toolkit. Moreover, the effect of training data scale on supervised versus reinforcement-based finetuning has not been systematically studied, particularly for out-of-domain generalization.

P — Proposal

The authors introduce HyperTrack, a dataset with over 16,000 tasks across 650+ Chinese mobile apps, and GUIEvalKit, an open-source toolkit for offline VLM benchmarking. They use HyperTrack to compare supervised and reinforcement-based finetuning across different data scales, revealing that reinforcement learning consistently outperforms supervised finetuning, with larger data amplifying the gap.

---

4) Experiments

Main results

Reinforcement-based finetuning achieves 72.3% task success rate on in-domain tasks and 58.1% on out-of-domain tasks, compared to 65.8% and 47.2% for supervised finetuning with the same data scale. Benchmarking SOTA VLMs (e.g., GPT-4V, Qwen-VL) shows best task completion of 68.5% on HyperTrack.

Ablation

Removing interaction history reduces task success by 12.4% on average across models, and ablating reasoning prompts (chain-of-thought) decreases performance by 9.7%, highlighting the importance of both components.

Limitations

HyperTrack focuses on Chinese mobile apps, limiting cross-cultural generalization. The toolkit currently supports offline evaluation only, not real-time interaction. The study uses a single VLM backbone for scaling experiments, so results may not fully transfer to other architectures.

5) Why it matters

  1. Provides a large-scale benchmark (HyperTrack) that can be used to evaluate your own VLM agents for mobile GUI tasks.
  2. Demonstrates that reinforcement-based finetuning is more data-efficient and generalizable than supervised finetuning, guiding training strategy choices.
  3. Offers an open-source toolkit (GUIEvalKit) for standardized offline evaluation, enabling reproducible comparisons.

1) 阅读理由

本文通过大规模数据集和统一评测工具，系统证明了基于强化学习的微调在移动GUI导航中优于监督微调，尤其在域外场景下优势显著。

2) CRGP（背景 / 相关工作 / 差距 / 方案）

背景：视觉语言模型（VLM）在移动GUI导航方面取得了进展，但现有数据集规模小且以英文为主，缺乏对中文应用的覆盖。数据规模对监督微调和强化微调下智能体性能的影响尚不明确，且缺乏标准化的评测工具。

相关工作：现有数据集如AITW（715k任务，357+应用，仅英文）、AndroidControl（15k任务，833应用，含边界框，英文）、AiTZ（2.5k任务，70+应用，含屏幕描述，英文）、AMEX（3k任务，192应用，含屏幕描述，英文）、GUI Odyssey（8k任务，212应用，含屏幕描述，英文）、CAGUI（600任务，22应用，中文，无边界框）等均存在局限。

差距：现有移动GUI数据集规模小且以英文为主，限制了评估的多样性和可重复性。缺乏对监督微调和强化微调下训练数据规模如何影响VLM智能体的系统研究，也缺少用于跨不同GUI任务进行一致基准测试的标准化工具包。

提案：作者引入了HyperTrack，一个包含超过16,000个真实世界任务、覆盖674个中文移动应用的大规模数据集，以及GUIEvalKit，一个用于统一离线基准测试的开源工具包。他们通过使用SFT和DAPO风格RL对UI-TARS-1.5-7B进行微调，跨越十个训练子集大小，系统研究了数据规模扩展，并评测了最先进的VLM以分析推理和交互历史的影响。

3) 实验

在HyperTrack上，使用8192个episode进行RL训练的模型在域内任务上达到了约95%的精确匹配准确率，而SFT约为90%。在未见过的应用任务上，RL达到了约88%的精确匹配，SFT约为80%。性能随训练数据量的对数近似线性增长。

消融实验

DAPO风格RL在相同数据规模下始终优于SFT，且差距在域外设置中扩大（例如，未见应用分割：RL约88% vs SFT约80%精确匹配）。用于点击操作的高斯空间奖励进一步提高了定位精度。

局限性

['HyperTrack专注于中文移动应用，限制了跨语言泛化能力。', '数据集仅从Android设备收集，未覆盖iOS或其他平台。', '规模扩展实验仅使用单一骨干网络（UI-TARS-1.5-7B），结果可能不泛化到其他架构。']

4) 重要性

  1. 提供了一个大规模的中文移动GUI数据集，可用于训练和评估VLM智能体在多样化真实世界任务中的表现。
  2. 展示了数据规模扩展与强化学习之间的协同作用，为提升智能体泛化能力提供了实用指导。
  3. 引入了一个开源评测工具包，标准化了基准测试流程，使未来工作能够进行公平比较。
