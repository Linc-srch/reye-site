Showboat Linux Malware Hits Middle East Telecom with SOCKS5 Proxy Backdoor

Showboat Linux Malware Hits Middle East Telecom with SOCKS5 Proxy Backdoor

Summary

Showboat is a modular Linux post-exploitation framework used since mid-2022 against a Middle Eastern telecom provider. It enables remote shell access, file transfer, and SOCKS5 proxy capabilities to pivot within internal networks. The malware, linked to Chinese threat actors including Calypso (Red Lamassu), shares code and infrastructure with other known backdoors like PlugX and ShadowPad. The initial access vector remains unknown, but past campaigns used ProxyLogon exploits or ASPX webshells. The findings underscore sustained cyber-espionage targeting critical telecommunications infrastructure, leveraging shared tooling to maintain persistent covert access.

摘要

Showboat是一个模块化Linux后渗透框架，自2022年中旬以来针对中东电信运营商使用，提供远程shell、文件传输及SOCKS5代理功能以在内网横向移动。该恶意软件关联中国APT组织（包括Calypso/Red Lamassu），与PlugX、ShadowPad等共享工具池。初始入侵向量未知，但过往活动使用ProxyLogon或ASPX webshell。研究凸显电信基础设施面临的持续间谍威胁。
