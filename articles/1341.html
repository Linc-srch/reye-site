GlassWorm Malware Takedown Disrupts Developer Supply Chain Attack Infrastructure

GlassWorm Malware Takedown Disrupts Developer Supply Chain Attack Infrastructure

supply-chain-attack zero-trust
Summary

CrowdStrike, Google, and the Shadowserver Foundation disrupted all command-and-control channels of GlassWorm malware, which since early 2025 targeted software developers via malicious VS Code extensions and npm/Python packages. GlassWorm used four resilient C2 methods: Solana blockchain, BitTorrent DHT, Google Calendar, and VPS infrastructure. It deployed a JavaScript RAT to steal credentials, crypto wallets, and browser data, and turned infected hosts into proxies and remote execution nodes. Over 300 GitHub repositories were poisoned. The takedown neutralized all channels simultaneously, preventing further instructions. The campaign is attributed to likely Russia-based cybercriminals.

摘要

CrowdStrike、Google和Shadowserver Foundation联合摧毁了GlassWorm恶意软件的所有command-and-control通道。自2025年初以来，GlassWorm通过恶意VS Code扩展和npm/Python包针对软件开发者，利用Solana区块链、BitTorrent DHT、Google Calendar和VPS四种C2方法实现弹性通信。该恶意软件部署JavaScript RAT窃取凭证、加密货币钱包和浏览器数据，并将受感染主机转化为代理和远程执行节点，污染了超过300个GitHub仓库。此次行动同时瘫痪所有通道，阻止了进一步指令下发。攻击者疑似俄罗斯网络犯罪分子。
