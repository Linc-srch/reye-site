2) CRGP

C — Context

Compositional speech-to-speech translation (S2ST) systems built on SpeechLLMs have shown promise, but typically encode source language as a flat label, ignoring cross-lingual structural regularities. This limits data-efficient adaptation when supervised multilingual S2ST data is scarce.

R — Related work

  • S2ST-Omni [32] introduces language-label conditioning into a SpeechLLM-based compositional S2ST framework.
  • Cascaded S2ST pipelines (ASR+MT+TTS) suffer from error propagation and lack global optimization.
  • End-to-end S2ST [23,21,4] and compositional S2ST [12,13,32] offer modular alternatives.
  • Typology-based language representations in NLP [7,29,36,31] motivate structured linguistic priors.

G — Gap

Existing S2ST systems either ignore source-language information or use flat language embeddings, which fail to capture systematic linguistic structures (morphology, reordering, genealogical relatedness) shared across languages. This oversight hinders data-efficient multilingual adaptation when supervised S2ST data is limited.

P — Proposal

S2ST-Omni 2 reformulates language conditioning into three structured levels: typology-informed hierarchical language encoding (TI-HLE) decomposes source language into morphology, reordering, genealogical family, and residual channels; dynamically-gated language-aware Dual-CTC modulates acoustic features adaptively; typology-aware LLM prompting injects linguistic priors into decoding. These components act as training-time inductive biases without increasing inference cost.

4) Experiments

Main results

On CVSS-C, S2ST-Omni 2 achieves average relative gains of 5.8% in BLEU and 4.6% in ASR-BLEU over S2ST-Omni, with consistent improvements in COMET and BLASER 2.0. A Japanese-to-English evaluation using only ~3 hours of supervised data shows explicit typological priors benefit data-efficient multilingual S2ST.

Ablation

Ablation studies confirm that representation-level (TI-HLE), acoustic-level (dynamically-gated LA-Dual-CTC), and decoding-level (typology-aware prompting) strategies provide complementary benefits, with each component contributing to overall performance.

Limitations

['The typological priors are hand-crafted and may not cover all relevant linguistic features across diverse languages.', 'Evaluation is limited to CVSS-C dataset; generalization to other domains or low-resource language pairs is not extensively validated.', 'The framework relies on a frozen Whisper encoder and Qwen3-4B decoder, which may not be optimal for all language pairs.']

5) Why it matters

  1. Demonstrates that structured linguistic priors (typology) can replace flat language labels for more data-efficient multilingual S2ST, relevant for low-resource language translation.
  2. Provides a modular framework (TI-HLE, Dual-CTC, typology-aware prompting) that can be adapted to other SpeechLLM-based systems.
  3. Shows that training-time inductive biases can improve performance without increasing inference cost, a practical advantage for deployment.

6) Next steps

  • [ ] Explore automatic extraction of typological features from raw speech or text to reduce manual engineering.
  • [ ] Extend evaluation to more language pairs and real-world noisy conditions to assess robustness.
  • [ ] Investigate integration of typological priors into other components (e.g., TTS backend) for end-to-end gains.
  • [ ] Apply similar structured conditioning to other multilingual tasks (e.g., ASR, MT) to validate generality.
