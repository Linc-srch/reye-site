Databricks brings GPT-5.5 to enterprise agent workflows

Databricks brings GPT-5.5 to enterprise agent workflows

2026-05-18 · openai_rss

Summary

Databricks integrates OpenAI's GPT-5.5 into enterprise agent workflows, achieving state-of-the-art performance on the OfficeQA Pro benchmark. The model reduces errors by 46% compared to GPT-5.4, excelling in parsing scanned PDFs, legacy files, and long-context documents. GPT-5.5 is available through AI Unity Gateway, AgentBricks, and Agent Supervisor API, enabling reliable orchestration of multi-step tasks without unnecessary detours. This advancement addresses critical parsing and retrieval challenges in production agent systems, marking a significant step-function improvement in enterprise knowledge work.

摘要

Databricks 将 OpenAI 的 GPT-5.5 集成到企业代理工作流中，在 OfficeQA Pro 基准测试上达到最先进水平。相比 GPT-5.4，错误率降低 46%，在解析扫描 PDF、遗留文件和长上下文文档方面表现优异。GPT-5.5 通过 AI Unity Gateway、AgentBricks 和 Agent Supervisor API 提供，能够可靠地编排多步骤任务，避免不必要的搜索绕路。这一进展解决了生产代理系统中的关键解析和检索挑战，标志着企业知识工作的阶跃式提升。
