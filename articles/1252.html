Microsoft Warns of Two Actively Exploited Defender Vulnerabilities

Microsoft Warns of Two Actively Exploited Defender Vulnerabilities

Summary

Microsoft disclosed two actively exploited vulnerabilities in Microsoft Defender: a privilege escalation flaw (CVE-2026-41091, CVSS 7.8) allowing SYSTEM privileges via improper link resolution, and a denial-of-service bug (CVE-2026-45498, CVSS 4.0). A third heap-based buffer overflow (CVE-2026-45584, CVSS 8.1) enabling remote code execution was patched but not exploited. The U.S. CISA added both exploited flaws to its Known Exploited Vulnerabilities catalog. Defender auto-updates address these issues; systems with Defender disabled are unaffected.

摘要

Microsoft披露了Defender中两个正被活跃利用的漏洞：权限提升漏洞CVE-2026-41091（CVSS 7.8）和拒绝服务漏洞CVE-2026-45498（CVSS 4.0）。第三个堆缓冲区溢出漏洞CVE-2026-45584（CVSS 8.1）虽已修补但未被利用。美国CISA已将前两个漏洞加入其Known Exploited Vulnerabilities（KEV）目录。Defender自动更新已修复这些问题，禁用Defender的系统不受影响。
