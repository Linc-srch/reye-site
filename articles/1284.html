Lazarus Deploys RemotePE Memory-Only RAT Against Financial and Crypto Firms

Lazarus Deploys RemotePE Memory-Only RAT Against Financial and Crypto Firms

content-management supply-chain-attack zero-trust
Summary

The Lazarus Group, linked to North Korea, has deployed a cross-platform memory-only RAT called RemotePE against financial and cryptocurrency firms. The attack chain uses DPAPILoader to decrypt RemotePELoader from disk, which then fetches RemotePE from a C2 server and executes it entirely in memory, leaving no filesystem artifacts. RemotePE supports six command categories for file operations, process management, and C2 configuration. The malware evades detection via Hell's Gate and ETW patching. Fox-IT found samples under active development from mid-2023 to mid-2024, indicating a purpose-built tool for long-term stealthy access and high-impact financial theft.

摘要

与朝鲜相关的Lazarus Group部署了名为RemotePE的跨平台内存-only RAT，针对金融和加密货币组织。攻击链使用DPAPILoader从磁盘解密RemotePELoader，后者从C2服务器获取RemotePE并在内存中执行，不留文件系统痕迹。RemotePE支持六类命令，用于文件操作、进程管理和C2配置。它通过Hell's Gate和ETW修补技术规避检测。Fox-IT发现该恶意软件在2023年中至2024年中处于活跃开发状态，表明其专为长期隐蔽访问和高影响金融盗窃而构建。
