Running Codex safely at OpenAI

Running Codex safely at OpenAI

2026-05-18 · openai_rss

Summary

OpenAI describes the safety measures for deploying Codex, a coding agent that autonomously reviews repositories, runs commands, and interacts with development tools. Key controls include sandboxing, approval policies, network access restrictions, identity management via ChatGPT, and rule-based command filtering. Agent-native telemetry via OpenTelemetry logs provides visibility into user prompts, tool decisions, and network events, enabling security teams to audit behavior and distinguish between expected actions and anomalies. The system balances developer productivity with security through managed configurations and AI-powered triage.

摘要

OpenAI 介绍了部署 Codex 的安全措施，该编码代理可自主审查代码库、运行命令并与开发工具交互。关键控制包括沙箱、审批策略、网络访问限制、通过 ChatGPT 的身份管理以及基于规则的命令过滤。通过 OpenTelemetry 日志的代理原生遥测提供了对用户提示、工具决策和网络事件的可见性，使安全团队能够审计行为并区分预期操作与异常。该系统通过托管配置和 AI 驱动的分类来平衡开发人员生产力与安全性。
