Gemini 3.1 Flash-Lite: Built for intelligence at scale

Gemini 3.1 Flash-Lite: Built for intelligence at scale

2026-05-18 · deepmind_blog

Summary

Google has launched Gemini 3.1 Flash-Lite, its fastest and most cost-efficient AI model, now available in preview via the Gemini API and Vertex AI. Priced at $0.25/1M input tokens and $1.50/1M output tokens, it offers 2.5x faster time to first token and 45% higher output speed than 2.5 Flash. It achieves strong benchmark scores including 86.9% on GPQA Diamond and an Elo of 1432 on Arena.ai. Designed for high-volume workloads, it supports tasks like translation, content moderation, UI generation, and simulations. Early adopters include Latitude, Cartwheel, and Whering.

摘要

Google 发布了 Gemini 3.1 Flash-Lite，这是其最快且最具成本效益的 AI 模型，现通过 Gemini API 和 Vertex AI 提供预览。定价为每百万输入 tokens $0.25、每百万输出 tokens $1.50，相比 2.5 Flash，首 token 响应速度提升 2.5 倍，输出速度提升 45%。该模型在 GPQA Diamond 上达到 86.9%，在 Arena.ai 上 Elo 评分为 1432。适用于翻译、内容审核、UI 生成和模拟等高容量工作负载。早期用户包括 Latitude、Cartwheel 和 Whering。
