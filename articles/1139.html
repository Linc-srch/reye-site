Work with Codex from anywhere

Work with Codex from anywhere

Summary

OpenAI announces that Codex, its AI coding agent, is now available in the ChatGPT mobile app as a preview. This allows users to stay connected to long-running coding tasks from their phone, reviewing outputs, approving commands, and providing guidance in real time. The feature uses a secure relay layer to sync state across devices without exposing machines to the public internet. It supports starting investigations, unblocking decisions, and synthesizing information on the go. Additionally, updates include Remote SSH support, programmatic access tokens, hooks for customization, and HIPAA-compliant use for enterprise workspaces.

摘要

OpenAI 宣布其 AI 编程代理 Codex 现已在 ChatGPT 移动应用中提供预览。用户可通过手机实时连接长时间运行的编码任务，审查输出、批准命令并提供指导。该功能使用安全中继层同步设备状态，无需将机器暴露于公共互联网。支持随时启动调查、解除决策阻塞和综合信息。此外，更新包括 Remote SSH 支持、程序化访问令牌、Hooks 自定义以及面向企业工作区的 HIPAA 合规使用。
