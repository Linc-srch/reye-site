1) Why-read

VIF introduces a lightweight module that continuously injects visual semantics during decoding to combat visual consistency decay in MLLMs, showing consistent gains across 14 benchmarks.

2) CRGP

C — Context

Multimodal large language models (MLLMs) typically use a connector-based paradigm where visual features are projected into the textual token space and concatenated as a prefix. This design treats visual tokens as static context, leading to diminishing visual influence as generation length increases.

R — Related work

  • BLIP-2 uses a Q-Former connector for vision-language alignment.
  • LLaVA employs a simple linear projection to map visual features.
  • InternVL-3 co-trains vision encoder and LLM for deeper semantic integration.
  • Qwen-LA uses reinforcement learning to insert image tokens into the output sequence.
  • VISTA introduces an alignment loss to keep output embeddings close to image representations.

G — Gap

Existing MLLMs suffer from visual consistency decay: as generation progresses, the model increasingly relies on linguistic priors and drifts from visual content. Prior work focuses on improving initial visual embedding quality or shallow alignment, but does not ensure sustained visual attention throughout the entire decoding process.

P — Proposal

The Vision Inference Former (VIF) is a lightweight module placed between the vision encoder and LLM that dynamically injects fine-grained visual features into the hidden states at each generation step. It consists of a self-attention layer, a cross-attention layer, and a fusion module, enabling continuous visual grounding without modifying the base model's architecture.

3) Figures

Figure extraction is performed separately and inlined post-generation.

4) Experiments

Main results

VIF improves LLaVA-1.5-7B from 70.2% to 72.8% on GQA, from 48.6% to 51.3% on TextVQA, and reduces hallucination on POPE from 85.4% to 87.2%. On MMBench, accuracy rises from 66.2% to 68.9%. Gains are consistent across LLaVA-1.5-13B and InstructBLIP-7B.

Ablation

Removing the cross-attention layer from VIF drops GQA accuracy from 72.8% to 71.1%, confirming that cross-modal interaction is critical for sustained visual grounding.

Limitations

['VIF adds a small number of parameters (approx. 1.5M), but inference latency increases slightly due to per-step cross-attention.', 'The module is designed for decoder-only LLMs; applicability to encoder-decoder architectures is not explored.', 'Evaluation is limited to English-language benchmarks; multilingual visual consistency is not tested.']

5) Why it matters

  1. Directly addresses the under-explored problem of visual consistency decay, which is critical for reliable MLLM deployment in safety-critical applications.
  2. The lightweight, plug-and-play design means it can be integrated into existing MLLMs without retraining the entire model, aligning with practical deployment needs.
  3. The continuous injection mechanism offers a new perspective on maintaining modality grounding during autoregressive generation, relevant to your work on system safety and AI alignment.
