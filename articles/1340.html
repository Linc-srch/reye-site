Malicious npm Package Stole Files From Claude AI User Directory via GitHub

Malicious npm Package Stole Files From Claude AI User Directory via GitHub

content-management supply-chain-attack zero-trust Malware & Threats Supply Chain Security
Summary

A malicious npm package named 'mouse5212-super-formatter' was discovered stealing files from Claude AI's user data directory. It authenticates to GitHub during installation, creates a repository, and uploads all files to a threat actor-controlled account. The package leaked its own GitHub token, suggesting AI-generated malware with poor operational security. It has been downloaded 676 times.

摘要

一个名为'mouse5212-super-formatter'的恶意npm包被发现从Claude AI的用户数据目录窃取文件。它在安装过程中通过GitHub认证，创建仓库并将所有文件上传到攻击者控制的账户。该包泄露了自身的GitHub令牌，表明这是AI生成的恶意软件且缺乏基本操作安全。该包已被下载676次。
