JINX-0164 Targets Cryptocurrency Firms with Fake Recruiter Lures and macOS Malware

JINX-0164 Targets Cryptocurrency Firms with Fake Recruiter Lures and macOS Malware

supply-chain-attack social-engineering
启示 · Insight

For a system security researcher focused on OS/kernel security and software supply-chain attacks, this campaign demonstrates a sophisticated multi-stage attack chain that bridges endpoint compromise (macOS malware) with developer infrastructure (CI/CD, npm). The use of macOS-specific payloads (Intel/Apple Silicon compatible) and lateral movement into code distribution systems highlights a growing threat surface: developer workstations as a pivot point into the software supply chain. The abuse of legitimate developer tools (npm packages) for backdoor delivery is a direct concern for supply-chain integrity.

对于专注于操作系统/内核安全和软件供应链攻击的系统安全研究员而言，该活动展示了一个复杂的多阶段攻击链，将端点攻陷（macOS恶意软件）与开发者基础设施（CI/CD、npm）相连接。使用macOS特定载荷（兼容Intel/Apple Silicon）并横向移动到代码分发系统，突出了一个日益增长的威胁面：开发者工作站作为进入软件供应链的支点。滥用合法开发者工具（npm包）进行后门投递直接关系到供应链完整性。

Summary

A previously undocumented threat actor, JINX-0164, targets cryptocurrency firms using fake recruiter lures on LinkedIn to deliver macOS malware (AUDIOFIX infostealer/RAT and MiniRAT backdoor), with lateral movement into CI/CD pipelines enabling supply chain compromise.

摘要

一个此前未被记录的威胁行为者JINX-0164，利用LinkedIn上的虚假招聘诱饵针对加密货币公司，投放macOS恶意软件（AUDIOFIX信息窃取器/RAT和MiniRAT后门），并通过横向移动到CI/CD管道实现供应链攻陷。

Deep Note

Key Points

  • Threat actor JINX-0164 targets cryptocurrency firms via LinkedIn recruiter lures.
  • Victims are tricked into downloading a Python-based macOS infostealer/RAT named AUDIOFIX.
  • AUDIOFIX steals credentials, SSH keys, crypto wallet data, and session tokens; supports remote shell and lateral movement.
  • Lateral movement targets CI/CD systems and internal code distribution, leading to a supply chain attack.
  • A second malware, MiniRAT (Go-based backdoor), was distributed via a compromised npm package (@velora-dex/sdk).
  • No infrastructure overlaps with known North Korean groups, despite tactical similarities.

Context & Contrast

This campaign is reminiscent of North Korean clusters (BlueNoroff, Contagious Interview) that use social engineering and crypto-theft, but Wiz explicitly states no infrastructure overlap. It differs by specifically targeting macOS with a custom Python RAT (AUDIOFIX) and by pivoting from endpoint compromise to supply-chain attacks via CI/CD. The use of a fake driver store domain and architecture-aware payloads is a notable technical detail.

Why It Matters

It underscores the convergence of endpoint malware and software supply-chain attacks, a critical concern for system security. The ability to move from a developer's macOS laptop into CI/CD pipelines and modify source code represents a high-impact attack vector. The use of macOS-specific malware and npm package compromise shows that supply-chain security must account for all platforms in the development toolchain.

Critical Take

Attribution to a specific state actor is unconfirmed; the lack of infrastructure overlap with known groups leaves open the possibility of a new financially motivated group or a false flag. The full technical details of the CI/CD lateral movement and source code modification are not publicly disclosed. The effectiveness of the social engineering lures is not quantified.

What to Watch

  • Public disclosure of AUDIOFIX and MiniRAT samples for technical analysis.
  • Any follow-up reports detailing the CI/CD lateral movement technique.
  • Whether similar fake recruiter campaigns appear targeting other sectors (e.g., fintech, SaaS).
  • Monitoring of npm for other compromised packages related to this actor.
  • Potential expansion of macOS malware targeting developer tools (e.g., Xcode, Homebrew).

背景与对比

该活动让人联想到使用社会工程和加密货币盗窃的朝鲜集群（BlueNoroff、Contagious Interview），但Wiz明确表示无基础设施重叠。其不同之处在于专门针对macOS使用自定义Python RAT（AUDIOFIX），并通过CI/CD从端点攻陷转向供应链攻击。使用虚假驱动商店域和架构感知载荷是一个值得注意的技术细节。

重要性

它强调了端点恶意软件与软件供应链攻击的融合，这是系统安全的一个关键问题。从开发者的macOS笔记本电脑移动到CI/CD管道并修改源代码的能力代表了一个高影响力的攻击向量。使用macOS特定恶意软件和npm包攻陷表明，供应链安全必须考虑开发工具链中的所有平台。

批判性视角

归因于特定国家行为者尚未确认；与已知组织缺乏基础设施重叠，留下了可能是新的经济动机团体或虚假旗帜的可能性。CI/CD横向移动和源代码修改的完整技术细节未公开披露。社会工程诱饵的有效性未被量化。

后续关注

  • AUDIOFIX和MiniRAT样本的公开披露以供技术分析。
  • 任何详细说明CI/CD横向移动技术的后续报告。
  • 是否出现针对其他行业（如金融科技、SaaS）的类似虚假招聘活动。
  • 监控npm中与该行为者相关的其他被攻陷包。
  • 针对开发者工具（如Xcode、Homebrew）的macOS恶意软件可能扩展。
