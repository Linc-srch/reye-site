Work with Codex from anywhere

Work with Codex from anywhere

2026-05-18 · openai_rss

Summary

OpenAI announces that Codex, its AI coding agent, is now available in the ChatGPT mobile app, allowing users to stay connected to long-running tasks from anywhere. The mobile experience provides full access to active threads, approvals, plugins, and project context via a secure relay layer. Users can start investigations, review outputs, approve commands, and change models on the go. The update also includes Remote SSH support for enterprise environments, programmatic access tokens, Hooks for customization, and HIPAA-compliant usage for healthcare. Codex in the mobile app is rolling out in preview on iOS and Android across all plans.

摘要

OpenAI宣布Codex现已集成到ChatGPT移动应用中，用户可通过手机随时跟进长期运行的AI代理任务。移动端提供完整的线程访问、审批、插件和项目上下文，通过安全中继层实现实时同步。用户可在通勤或会议间隙启动调试、审查输出、批准命令或切换模型。更新还包括Remote SSH支持企业环境、Programmatic Access Tokens、Hooks自定义功能以及面向医疗的HIPAA合规使用。Codex移动预览版已在iOS和Android上向所有计划用户开放。
