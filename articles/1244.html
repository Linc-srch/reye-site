Ghostwriter Targets Ukraine Government Entities with Prometheus Phishing Malware

Ghostwriter Targets Ukraine Government Entities with Prometheus Phishing Malware

Summary

The Belarus-linked threat actor Ghostwriter (UAC-0057) is targeting Ukrainian government entities with phishing emails disguised as notifications from the Prometheus online learning platform. The campaign, active since spring 2026, uses compromised accounts to deliver a PDF with a link to a ZIP archive containing a JavaScript file (OYSTERFRESH). This file drops an obfuscated payload (OYSTERBLUES) into the Windows Registry, which harvests system information and communicates with a C2 server. The final payload is Cobalt Strike, used for post-exploitation. CERT-UA advises restricting wscript.exe execution for standard users to mitigate the threat.

摘要

与白俄罗斯相关的威胁组织Ghostwriter（UAC-0057）利用乌克兰在线学习平台Prometheus相关的诱饵，通过钓鱼邮件攻击乌克兰政府实体。该活动自2026年春活跃，使用被攻陷的账户发送包含PDF附件的邮件，附件链接指向ZIP压缩包内的JavaScript文件（OYSTERFRESH）。该文件将混淆加密的有效载荷OYSTERBLUES写入Windows注册表，用于收集系统信息并通过HTTP POST发送至C2服务器。最终载荷为Cobalt Strike，用于后渗透活动。CERT-UA建议限制标准用户运行wscript.exe以降低风险。
